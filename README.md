# PMI GYM - Comprehensive Gym Management System 🏋️‍♂️

## Overview
PMI GYM is a feature-rich desktop application designed to manage gym operations and track user fitness progress. It combines a Tkinter-based Graphical User Interface (GUI) with a robust logic backend to simulate a real-world gym management environment. 

## Key Features ✨
* **User Profiles & BMI:** Automated registration with BMI and BMR calculations.
* **Smart Planning:** Customized generation of training routines (Bulking, Cutting, Cardio) and corresponding daily nutrition limits.
* **Data Logging:** Local database system utilizing JSON and text files to track daily caloric intake and store purchases.
* **Loyalty & Store System:** Interactive gym store with a gamified loyalty points system.
* **Utility Tools:** Dynamic Fatigue Score Calculator based on sleep and workout intensity, plus a daily progress tracker.

## Tech Stack 🛠️
* **Frontend/GUI:** Python (Tkinter)
* **Backend Logic:** Python 
* **Data Storage:** JSON, Local File I/O

## How to Run 🚀
1. Clone this repository.
2. Ensure you have Python 3.x installed.
3. Run the main script: `Gym project.py`
